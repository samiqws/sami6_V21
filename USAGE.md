# Quick Start Guide

## Installation

1. Run as Administrator:
```bash
install.bat
```

2. Start the system:
```bash
start.bat
```

3. Access dashboard: http://localhost:8000

## Configuration

Edit `config/settings.json` to customize:
- Protected paths
- Detection sensitivity
- Containment actions
- Decoy settings

## Testing

Create test files in protected directories to see monitoring in action.

**Warning**: Do not test with real ransomware!
